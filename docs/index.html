职场上没有坏人，也没有好人。

开始实习!!

  1. 准备实习简历

  4. 做好准备，提供选项给对方

实习要点

  1. 要干实事，不然宁可换工作
  2. 寻求转正的机会
  3. 为寻找下一份工作做准备

第一份工资

税前工资

一般来说，我们和企业商定的是“税前工资”，发放前会扣除社保和公积金，然后扣除个人所得税，发到手的是税后工资。

按照规定，我们每月的工作时间是 21.75 天，请假、缺勤都以此来计算。

社保

通常来说，我们会缴纳五份保险：

  1. 养老保险
  2. 医疗保险
  3. 失业保险
  4. 工伤保险
  5. 生育保险

养老保险

  1. 由个人(9%)和公司(22%)一起缴纳
  2. 有上下限
  3. 在同一地区缴满10~15年即可办理退休

我国养老保险分两部分组成，第一部分：

  1. 一种国家强制储蓄
  2. 从工资里强制扣除，存入个人养老账户，等退休之后发放

如果只有这一部分，会遇到很严重的问题：

  1. 随着经济发展，社会总财富在增加，货币在增加
  2. 由于通货膨胀，几十年后，货币的价值会严重缩水

举例：我小时候，我爸妈的工资30~40块，现在广州社平工资7k+

所以，养老的第二部分：从现有保险中，提取一部分（社会统筹）发给退休人员，保证他们的生活。

前提：假定始终有足够的人缴纳养老保险。

推论：

一定要待在大城市里，不然自己的后代迟早面临挤入大城市的挑战。

对于我来说，即使做全职远程，我也要至少待在广州。

注意事项：

  1. 不要断缴，要知道如何自己查询。实在没有下家，找淘宝。
  2. 关注下缴费基数，缴费比例，强制执行，可操作空间不大。
  3. 关于异地转出转入，只能自己跑跑社保中心。每个地方细则不一样，社保有很多办事指南，具体问题具体解决。

医疗保险

  1. 由个人(2%)和公司(10%)一起缴纳
  2. 有下限
  3. 使用有上限

注意事项

  1. 非常划算好用，总体上入不敷出，以后退休人员也要缴纳医保。
  2. 地区不一样，医疗执行有些许差别。北京每月会返现金，用途不限；广州每月有门诊费用；其它地方药店卖洗衣粉大米，不过涉嫌骗保，以后会严管。
  3. 住院，大约报销总额60-70%，总有些自费项目药品，医院需说明。慢性病比如糖尿病还有专门补贴。
  4. 不足之处，大病疑难病，保费不足。比如进一次icu，报销比例骤降，撑不了几天。癌症的靶相药基本全自费。
  5. 建议自行补足商业保险。

失业保险和工伤保险

  1. 用处不大
  2. 失业只有被辞退才能用
  3. 程序员工伤率比较低

生育保险

  1. 公司缴纳
  2. 分为，产前检查和分娩费用
  3. 使用有上限和项目限制

注意事项：

  1. 生育津贴会返到公司账户，所以还是从公司领
  2. 需要大量证明材料和琐碎的手续，请及时跟公司HR要材料，居委会大妈打好关系

住房公积金

  1. 由个人(12%)和公司(12%)一起缴纳
  2. 100%计入个人账户，取用方便，避税的好办法
  3. 公积金贷款利息远低于商贷，和缴费额度有关，所以能多交就多交

注意事项：

  1. 非常好的福利。基数缴的越高越赚。因为里边的钱都是你的。还能避税。
  2. 提取，无非是买房、租房、装修房，以及离职。公积金管理相对比较松，因为钱都是个人的，相当于银行存款。他不能擅自挪用，所以什么时候去取，只要有名目，都可以。比如，生病了急需用钱之类的。
  3. 买房提取就不说了，一般来说大家愿意用公积金贷款，而不是提取现金。
  4. 租房提取，以前还要租房合同和发票。现在考虑到实际情况，北京已经啥都不要了，每月1500上限。只要填个表就行，人性化了不少。其它地区咨询当地政策。
  5. 离职提取，也很简单，离职时候跟公司说一声，直接全部提走，广州地区可以操作。北京地区暂时不详，以前是不可以的。

个税

  1. 公司代缴，可操作空间不大，数值网上很多个税计算器，找新的问题不大
  2. 扣完五险一金之后再扣
  3. 年收入超过12万需个人申报，只是申报，并不多扣

第一次离职

小宅工作努力认真负责，使得公司在付费专题方面取得大幅进步。这当然是件好事，每次调薪名单里都有小宅，比起刚刚加入公司，他的工资整整翻了2番。

职场小窍门

评估自己的薪资

职场上问工资是大忌，这里面有几方面原因：

  1. 公司要平衡大家的薪资，但是影响工资的原因非常多，很难做到真的平衡，所以保持神秘感也是必然的选择
  2. 人们总是倾向于“我是最吃亏的那个”，所以打听到别人工资的具体数目时，很少会有人感到满足
  3. 对个人而言，问收入也很不礼貌

那我们是不是就无法评估自己的薪资呢？

答案当然是否定的。

衡量自己工资几法：

  1. 已交结成特别好友，私下里直接问
  2. 一般好友，不经意间泄露自己的模糊薪水，期待对方给与回应
  3. 通过招聘网站评估

不建议的方法：

  1. 在论坛问
  2. 偷看别人的工资单

职场难题二

提出离职，受到挽留，是否可以留下呢？

  • 短答案：不要。

  • 长答案：如果裸辞，可以考虑一下；不然的话，不要。

理由：

  1. 提出离职，会破坏雇工双方的信任
  2. “被迫”加薪会影响公司的薪资结构，影响将来薪资调整
  3. 通过稳定团队来维护部门战斗力不是普通员工的职责
  4. 没有谁离不开谁，你的岗位很多人在觊觎

职场难题三

提出离职，领导不批准怎么办？

  1. 稳定而坚决的告诉对方，自己一定要离职
  2. 更大范围的提交辞呈，让更多的人知道
  3. 不接新任务，只处理文档准备交接

从法律上来讲，提交辞呈后，需要一段时间交接工作，通常不超过一个月。之后，便是自由人，公司没有权力强行留下一个人。

职场必修课

如何离职

  1. 拒绝新工作新任务，只处理以前的 bug
  2. 尽可能多留下文档，跟接班人交接
  3. 拷走私人文件、删除私人内容、登出各种账号
  4. 休完年假，处理好报销、借款等
  5. 和 HR 确定最后一天，其间准时上下班
  6. 办手续领取离职证明，确认是否启动竞业禁止
  7. 处理社保

职场必修课

维持社保连续

很多大型城市都把“连续缴纳社保N个月”作为享受各项福利的依据：

  1. 买房
  2. 汽车摇号
  3. 证件异地办理

所以，

对于现代外漂青年来说，保持社保联系非常必要!

  1. 每个城市都有“扣缴社保日”
  2. 如果“扣缴日”在你的离职入职之间，那么就有风险漏缴
  3. 此时你可以跟原公司协商晚一点减员，或跟新公司协商早一点增员，或淘宝代缴
  4. 亦可以到新公司立刻查补（目前大部分城市补缴有效）
  5. 如果跨年，不要等着查补，不能跨年补缴

提醒：劳动关系发生变更，不仅限于离职跳槽，不过内部调岗，子公司变动，公司法人名称变动等，都是断缴高发期，自己一定要注意，及时查及时补。

城市管理者在制定政策时会参考当前值，所以这些指标都是临时、多变的，并且政策会传播。所以暂时不受限制的同学也不妨多注意，已经被限制的更是越多越好。

一个猜想，不一定对。

职场必修课

为什么不要进外包公司

  1. 很难接触到核心产品核心技术
  2. 没有自己的产品，没有长期持续的投入
  3. 无法和甲方取得共同成长
  4. 限于人员成本和企业投入，个人成长也有限

职场必修课

好工作的标准是什么

  1. 收入。尽可能高，能让人体面地生活
  2. 工作内容。既符合个人兴趣，又能创造真正的价值
  3. 发展前景。无论是公司还是个人，都要有较好的发展前景
  4. 工作环境。舒适，同事之间相处融洽，工作“有尊严”
  5. 守法。不触犯法律，也不要处在灰色地带

对比小宅这次跳槽，我们发现，他只关注了收入的提升和降低工作压力，找到了一份工作内容差、发展前景差的工作，得不偿失。

最终他会为这个草率的决定付出代价。

（回到故事中……）

又过了两天，前领导打电话来，说想请小宅回去。小宅几乎立刻就被说动了。

于是，经过短暂的请辞、交接、离职，小宅回到了知春路上，那个熟悉的某某大厦。

时过境迁

这次离职前后大概5个半月，但实际情况已经发生了变化：

  1. 小宅原来的制作组升级成前端组，人员由4人扩编到8人
  2. 前端组由新的技术经理管理
  3. 产品经理成为互联网公司标配，公司里的产品部门风生水起
  4. 老板委任小宅为“商业产品经理”，主管商业产品，和原产品部门并行

小宅本以为回来能继续前面的工作，没想到整个环境都大变样，不过既回来之则安之，先干着再说吧。

很快，他发现新的工作并不好做。

职场必修课

公司是活着的生物

公司不仅是松散的人类集合，它已经是一个独立的活物

  1. 人饿了，就会想吃东西；人渴了，就会想喝水。公司也一样，不同组织部门会驱动公司按照他们的需求做事。
  2. 人的行动，一部分受大脑控制，还有一部分受小脑控制。公司也一样，很多部门组织都会自发行动。
  3. 人的肌体，会有本能反应，比如排异，不受大脑空知。公司也一样，每个人每个部门都有小群体利益，老板也解决不了。

这样，我们就得动态的理解小宅的处境：

  1. 他之前是制作组组长；然而制作组已经升级成前端组，并有新的技术经理负责
  2. 他不再负责商业专题，却想规范之，必然受到整个流程上各个环节的抵制
  3. 老板作为大脑，当然有自己的想法；但是想贯彻下去，就需要更多的支持

于是，小宅陷入新角色的挣扎。

他原本想象自己踏着彩云归来，以救世主的形象从天而降，没想到大家根本不怀念他，甚至原以为是朋友的人也站到自己的对立面。

一切都说明，小宅的位置，坐不久了。

更令小宅不爽的是，入职之前，只说好做“商业产品经理”，但级别没定下来。

于是逢年过节发福利的时候，小宅只能按照新人的标准领。

小宅去找行政，行政说：啊呀不好意思，不过我们这里没有接到通知，要不你去找一下老板？

小宅想，为这千把块钱去找老板，似乎有点兴师动众；再加上自己的业务不很出彩，算了吧……

小宅索性不再跟大家争来争去，转头去做自己认为值得做的产品。虽然屡获好评，但最终只落个不务正业的评价。

失之东隅，收值桑犁，小宅的需求嗅觉、产品品味、实现能力被一个朋友相中，邀请他加入创业公司。

于是，新的生活展开了。

投身创业

这次离职顺利得多，可能双方都庆幸能够摆脱多方吧。

随后，小宅加入了朋友的创业公司，任产品总监职务。

实际上，因为新公司 CTO 已有人选，所以“产品总监”只是个虚名。

离职前，一位朋友问小宅：这个老板靠谱么？

小宅想了想说：

  1. 人不能用靠不靠谱来衡量，应该问，这个人，看到多少钱会动心
  2. 这个老板，出身很好，创业对他来说是实现个人价值的机会，相信他不会为了一点蝇头小利乱来
  3. 亲兄弟明算帐，一切约定走协议，不相信口头承诺

小宅发现，加入创业公司的体验非常新奇，比如，股票和期权是什么？

职场必修课

股票和期权

起因

普通员工到了一定阶段，会缺乏动力，因为再怎么做也很难有更高的回报了。

所以聪明的公司选择发放股份和期权给员工，这样员工就能从分红和股价上获取财务回报，继而会更加努力的工作。

另一方面，创业公司财力有限，开不出那么有竞争力的工资，也会选择用股票和期权这样的长期激励争取人才。

股票

  1. 股票 ≠ 投票权
  2. 股票不一定能直接交易
  3. 股票交易需要交税
  4. 基本上相当于现钱

股份

  1. 授予股份需要做股份变更
  2. 能够在工商网站上查到
  3. 需要承担公司的责任

期权

  1. “在一定时间内，可以以行权价的买入股票”
  2. 基本上公司会直接回购
  3. 同样需要交税
  4. 通常分批发放

注意事项

  1. 股票期权，都不是现金，都存在风险
  2. 变现周期非常长
  3. 建议作为财务回报的补充，而不是主体

（回到故事中……）

小宅因为家庭原因，选择最少的期权和较高的工资。

换言之，放弃长期财务回报，追求短期回报。

同时，以较低系数缴纳五险一金，同时避税。

小宅发现，对比之前较成熟的公司，创业公司有以下不同：

  1. 对个人的要求更广，不需要螺丝钉，要工具箱
  2. 更依赖老板的眼光，老板看得准，公司就走得好
  3. 没有技术负债，做起东西来更自由灵活

买房

工作这么久，小宅攒了一些钱，跟老婆一商量，准备买套房。

早年小宅一点都不想要家里的钱，如今他的想法变了。

  1. “政府天然有滥发货币的倾向。” => 爹妈的存款会越来越不值钱
  2. 社会上有各种针对老年人的骗局，基本总有一款适合你
  3. 父母一代并不乐衷于消费

如果完全拒绝家里帮助：

  1. 几乎没有条件上车
  2. 只能买很差的房子，自住投资都亏
  3. 父母的钱会贬值，也有可能被骗……

关于房地产，他是这么看的：

  1. 房地产不会永远上涨，但是大城市只要保持活力，房地产就不会跌
  2. 没钱不用刻意追求，但有点现金，买房是最好的保值手段
  3. 考虑到将来变现，买房尽量买到好地段好户型好朝向
  4. 买房时，建议寻找热心亲友陪同

融资 & 合规

公司运营状况不错，于是老板开始准备A轮融资。

职场必修课

融资

创业公司往往自造血能力不足，或者竞争激烈，为了争夺时间窗口，大部分创业公司都会融资。

融资从天使开始，一般以ABCD作为编号。

每一轮融资有新投资人加入，也会有老投资人离开。

融资未必会立刻全额到帐，和回款一样，可能会分批，有数据要求。

所以融资成功并不意味着高枕无忧。

融资一般伴随着敬业调查。

（回到故事中……）

很快，HR 找到小宅谈话。公司将来肯定要上市的，所以从现在开始就要准备做合规。

所以不能再避税，五险一金也要足额缴纳。

小宅表示，只要公司能够成长，这样也可以接受。

创业失败

行百里者半九十，最终，这个创业公司也没能走到最后。

裁员很早就开始了，一批批的同事离开，换来一封封的离职通知。

终于，轮到小宅了。

HR 找到小宅，说：公司因为经营问题，决定裁掉他的部门，

  1. 补偿 N+1
  2. 不启动竞业禁止

职场必修课

辞退补偿

无论是合同到期不续签，还是辞退员工，都需要给予补偿。

补偿标准通常是 N+1（有一个月的交接）和 2N（立即离开）。

这里的 N 是在本公司的工作年限，不到半年算 0.5；过半年不满一年算一年。

算出的结果乘以月薪既补偿金。

这里存在上线，N上限12，月薪上限当地社平工资*3。

（回到故事中……）

小宅对补偿结果不满意，毕竟如果按社平工资的三倍补偿的话，补偿总额比按工资少很多。

HR 说：

  1. 这个是公司决定，不是为难小宅，也不会为小宅修改
  2. 公司现况不好，拿出这笔钱也不容易，建议小宅接受
  3. 这个方案完全合法，仲裁也没用
  4. 如果不接受，人力部门也有一些手段，到时候两边都不好看

小宅当然很失望，但他已经不是当初那个懵懂少年，他自己核算：

  1. 公司肯定是不行了，早死晚死的区别
  2. 今天走，快速拿到补偿金
  3. 今天不走，也许补偿金会增加，但是，自己要浪费时间；而且，万一拖到公司垂死，拿不出钱更糟糕

于是他接受了这个方案。

接下来干嘛呢？小宅也不知道。

不过，经历过以上种种，小宅相信自己能够兵来将挡水来土掩。

哦，对了，在老婆的建议下，小宅去办理了公积金提取。

根据广州市政策，外地人离职可以办理公积金提取，很方便，凭借离职证明即可。

其它城市请参考本地政策。

其它常见问题

不好放在故事里，单独解答一下吧。

在公司授意下，程序员是否可敲写‘灰色’代码？

答：

法律上有一个明确的原则，叫“不得违反上位法。”

所以“公司可以给我一个授权，说此为公司强制要求行为，和我无关”这个承诺是废纸一张，哪怕你的领导勇敢的站出来，这个锅他也背不上（当然有别的锅给他）。

所以，在职场上，千万不要干违法的事儿。一旦被查实，将来背上前科，基本所有问题都要自己承担。

异地社保转入需要什么流程？

答：

需要查阅转出地和转入地的政府文件，每个地方都不太一样，这里不再赘述。

公积金一直有个人账户，建议直接销户提取。

社保会有一定的损失，且不算连续，个人建议不转也罢。

个人的短期理财和长期理财计划？

答：

我实际不擅长理财。

建议：办信用卡，理性消费

建议不要：用小贷

如何认识加班

答：这个问题之前提过很多次。加班的目的：

  1. 为了完成工作，而完成工作是集体的事情，所以要积极寻求帮助
  2. 不要为加班而加班，如果一个公司/部门把加班作为积极信号，离开比较好
  3. 加班时，如果能够争取对自己有益的权利，不要忘记争取
